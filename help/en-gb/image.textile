h1. Textile: Images

To insert an image, put the URL for the image inside exclamation marks.

*Optional:* text that immediately follows the URL in (parentheses) will be used as the @alt@ text for the image (this also adds a @title@ attribute with the same content). Images on the web should always have descriptive @alt@ text for the benefit of readers using non-graphical browsers.

*Optional:* place a colon followed by a URL immediately after the closing @!@ to make the image into a link.

h3. Examples

bc. !http://textpattern.com/assets/img/textpattern.png(Textpattern icon)!

Will become:

bc. <img src="http://textpattern.com/assets/img/textpattern.png" alt="Textpattern icon" title="Textpattern icon" />

Which, when rendered by a browser, will appear like this:

!http://textpattern.com/assets/img/textpattern.png(Textpattern icon)!

With a link:

bc. !http://textpattern.com/assets/img/textpattern.png(Textpattern icon)!:http://textpattern.com

Will become:

bc. <a href="http://textpattern.com"><img src="http://textpattern.com/assets/img/textpattern.png" alt="Textpattern icon" title="Textpattern icon" /></a>

Which, when rendered by a browser, will appear like this:

!http://textpattern.com/assets/img/textpattern.png(Textpattern icon)!:http://textpattern.com
