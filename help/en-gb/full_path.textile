h1. Preferences: Path to Textpattern

This must indicate the full server path to your @/textpattern/@ directory, which should be inside the directory containing your Textpattern site.

p(alert-block information). *Note:* There must be no trailing slash at the end of the path.

h3. Examples

bc. /home/user/docs/html/textpattern

or

bc. /home/var/www/textpattern
